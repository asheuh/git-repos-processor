# repos clone script

## Application installation

- Virtual Environment

```
$ python3 -m venv venv
$ source venv/bin/activate
```

- Dependencies

```
$ pip install -r requirements.txt
```

## Try it

- Clone from file
```
$ python clone.py file repos.txt
```

- Clone user repos
```
$ python clone user your-github-username
```

- For help
```
$ python clone.py --help
```
